## How to run the app locally?

```
# install yarn
npm install --global yarn

# install client app dependencies
yarn

# run client app
yarn run dev

# go to server app
cd express

# install server app dependencies 
yarn

# run server app
yarn run dev
```

Voila! The app is available at `locahost:8100`.
